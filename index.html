What is a Servo Motor?

Definition

A servo motor is a special type of motor that can rotate to a specific angle and hold that position.

Key Characteristics

  • Precise Control: Can move to exact angles (0¬∞ to 180¬∞)
  • Position Feedback: Knows where it is and corrects itself
  • Three Wires: Power, Ground, and Control signal
  • Small Size: Perfect for robotics and automation

How is it Different from Regular Motors?

Regular DC Motor

  • Spins continuously
  • Speed control only
  • No position feedback
  • Two wires (+ and -)

Servo Motor

  • Moves to specific angles
  • Position control
  • Built-in feedback system
  • Three wires (Power, Ground, Signal)
üí° Fun Fact: Servo motors are used in RC cars, robots, camera gimbals, and even aircraft control surfaces!

Why These Pins?

Digital Pin 9 (PWM)

  • PWM Capable: Can generate precise timing signals
  • Servo Library Support: Optimized for servo control
  • Alternative Pins: 3, 5, 6, 9, 10, 11 (all PWM pins work)

5V Power Pin

  • Stable Voltage: Provides consistent 5V power
  • Sufficient Current: Can handle small servo motors
  • Safe: Protected by Arduino's voltage regulator

Step-by-Step Wiring Guide

What You Need

  • Arduino Uno board
  • Small servo motor (SG90 or similar)
  • 3 jumper wires
  • USB cable for Arduino
  • Computer with Arduino IDE

Wiring Steps

Step 1: Identify Servo Wires

Wire Color Function Connect To
Red Power (+5V) Arduino 5V pin
Brown/Black Ground (0V) Arduino GND pin
Orange/Yellow Control Signal Arduino Pin 9

Step 2: Make Connections

  1. Power First: Connect RED wire to 5V pin
  2. Ground Second: Connect BROWN/BLACK wire to GND pin
  3. Signal Last: Connect ORANGE/YELLOW wire to Digital Pin 9
  4. Double Check: Verify all connections are secure
‚ö†Ô∏è Safety First: Always disconnect power when making wiring changes!

Programming Concepts

Servo Library

Arduino provides a special Servo library that makes controlling servos very easy!

Key Functions

  • attach(): Connect servo to a pin
  • write(): Move servo to specific angle (0-180¬∞)
  • read(): Get current servo position
  • detach(): Disconnect servo from pin

PWM Signal Explanation

Servos use PWM (Pulse Width Modulation) signals:

  • Frequency: 50 Hz (one pulse every 20ms)
  • Pulse Width: 1-2ms determines angle
  • 1ms pulse: 0¬∞ position
  • 1.5ms pulse: 90¬∞ position (center)
  • 2ms pulse: 180¬∞ position

Basic Program Structure

#include <Servo.h> // Include servo library Servo myServo; // Create servo object void setup() { myServo.attach(9); // Attach servo to pin 9 } void loop() { myServo.write(90); // Move to 90 degrees delay(1000); // Wait 1 second }
üí° Remember: The servo library handles all the complex PWM timing for you!

Code Example 1: Basic Movement

Simple Back and Forth Movement

Let's start with a simple program that moves the servo back and forth:

// Include the servo library #include <Servo.h> // Create a servo object Servo myServo; void setup() { // Attach the servo to pin 9 myServo.attach(9); // Initialize serial communication for debugging Serial.begin(9600); Serial.println("Servo Test Started!"); } void loop() { // Move to 0 degrees Serial.println("Moving to 0 degrees"); myServo.write(0); delay(1000); // Wait 1 second // Move to 90 degrees (center) Serial.println("Moving to 90 degrees"); myServo.write(90); delay(1000); // Wait 1 second // Move to 180 degrees Serial.println("Moving to 180 degrees"); myServo.write(180); delay(1000); // Wait 1 second // Move back to center Serial.println("Moving back to 90 degrees"); myServo.write(90); delay(1000); // Wait 1 second }

Code Explanation

  • Line 2: Include servo library
  • Line 5: Create servo object named "myServo"
  • Line 8: Attach servo to pin 9
  • Line 15, 20, 25, 30: Move servo to different angles
  • delay(1000): Wait 1000 milliseconds (1 second)
üí° Try This: Change the delay values to make the servo move faster or slower!

Code Example 2: Smooth Sweep

Gradual Movement

This code makes the servo move smoothly from 0¬∞ to 180¬∞ and back:

#include <Servo.h> Servo myServo; void setup() { myServo.attach(9); Serial.begin(9600); Serial.println("Smooth Servo Sweep Started!"); } void loop() { // Sweep from 0 to 180 degrees for (int angle = 0; angle <= 180; angle++) { myServo.write(angle); Serial.print("Angle: "); Serial.println(angle); delay(15); // Small delay for smooth movement } // Wait at 180 degrees delay(500); // Sweep from 180 to 0 degrees for (int angle = 180; angle >= 0; angle--) { myServo.write(angle); Serial.print("Angle: "); Serial.println(angle); delay(15); // Small delay for smooth movement } // Wait at 0 degrees delay(500); }

New Concepts

  • for loop: Repeats code with changing values
  • angle++: Increases angle by 1 each time
  • angle--: Decreases angle by 1 each time
  • delay(15): Very short delay for smooth motion
üí° Experiment: Try changing delay(15) to delay(50) - what happens?

Code Example 3: User Control

Control Servo from Serial Monitor

This code lets you control the servo by typing angles in the Serial Monitor:

#include <Servo.h> Servo myServo; int currentAngle = 90; // Start at center position void setup() { myServo.attach(9); Serial.begin(9600); // Move to starting position myServo.write(currentAngle); Serial.println("Servo Control Ready!"); Serial.println("Enter angle (0-180) and press Enter:"); Serial.print("Current angle: "); Serial.println(currentAngle); } void loop() { // Check if data is available to read if (Serial.available() > 0) { // Read the incoming angle int newAngle = Serial.parseInt(); // Check if angle is valid (0-180) if (newAngle >= 0 && newAngle <= 180) { currentAngle = newAngle; myServo.write(currentAngle); Serial.print("Moved to angle: "); Serial.println(currentAngle); } else { Serial.println("Error: Please enter angle between 0 and 180"); } Serial.println("Enter next angle:"); } }

How to Use

  1. Upload the code to Arduino
  2. Open Serial Monitor (Tools ‚Üí Serial Monitor)
  3. Type any number from 0 to 180
  4. Press Enter
  5. Watch the servo move!

Code Example 4: Multiple Servos

Controlling Two Servos

This example shows how to control multiple servos simultaneously:

#include <Servo.h> // Create two servo objects Servo servo1; Servo servo2; void setup() { // Attach servos to different pins servo1.attach(9); // First servo on pin 9 servo2.attach(10); // Second servo on pin 10 Serial.begin(9600); Serial.println("Dual Servo Dance Started!"); // Start both servos at center position servo1.write(90); servo2.write(90); delay(1000); } void loop() { // Synchronized movement - both move together Serial.println("Synchronized movement"); for (int angle = 0; angle <= 180; angle += 5) { servo1.write(angle); servo2.write(angle); delay(50); } delay(500); // Opposite movement - mirror each other Serial.println("Mirror movement"); for (int angle = 0; angle <= 180; angle += 5) { servo1.write(angle); servo2.write(180 - angle); // Opposite direction delay(50); } delay(500); // Return to center servo1.write(90); servo2.write(90); delay(2000); }
